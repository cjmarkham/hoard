Event and data storage with embedded query system built ontop of MongoDB
===

### Install

Console:

	php ./install/shell.php \
		--mongodburi=mongodb://admin:admin@127.0.0.1:27017/hoard \
		--docroot=/path/to/install \
		--email=admin@example.com \
		--deleteinstall=true
